Descripcion de la pagina:
Esta página web es un catálogo informativo sobre cuatro títulos icónicos de la industria de los videojuegos: Left 4 Dead 2, Batman: Arkham City, Hollow Knight y Phasmophobia. El sitio permite navegar a través de secciones específicas para conocer la trama, los desarrolladores y la cronología o mapas de cada juego, utilizando un diseño limpio y un sistema de navegación mediante menús desplegables

vinculacion:
Se conectó el estilo mediante <link rel="stylesheet" href="assets/CSS/style.css"> para separar el diseño de la estructura HTML.

Gestión de Imágenes en el CSS:
Sección Left 4 Dead: Se usaron los selectores :first-child (49%) y :last-child (45%) para dar medidas distintas a las fotos del mismo contenedor.
Resto de Juegos: Se aplicaron clases directas con anchos fijos de 460px, 900px y 510px para asegurar que cada imagen tenga una dimensión única en el sitio.
Proporción: Se aplicó height: auto para evitar que las imágenes se deformen al cambiar su ancho.
